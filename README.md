# SpendSmart – Expense Tracking Web Application

LIVE DEMO:https://drive.google.com/drive/folders/1HdQ24bAzvhrIOQ4MdNUcgyOhVMSAy8sY?usp=sharing

SpendSmart is a web-based expense tracking application developed and deployed using **Replit Cloud**. The application helps users manage their personal finances by tracking income and expenses through a simple and interactive web interface.

Replit Cloud acts as a complete online Integrated Development Environment (IDE), allowing the application to be developed, executed, and hosted entirely in the cloud without requiring any local setup. By leveraging Replit Cloud’s real-time collaboration, instant deployment, and persistent database features, SpendSmart provides an efficient and accessible platform for both development and hosting.

---

## Features

- User registration and secure login
- Add, view, and manage income and expense records
- Interactive and user-friendly web interface
- Persistent data storage using a lightweight database
- Cloud-based deployment with no external dependencies

---

## System Overview

SpendSmart uses a **Flask-based backend** to handle application logic and database operations. The frontend is built using standard web technologies to provide a responsive and intuitive user experience. Replit Cloud simplifies deployment, database management, and application hosting, enabling continuous development and real-time updates.

---

## Architecture

- **Backend:** Flask (Python)
- **Database:** SQLite integrated using SQLAlchemy ORM
- **Frontend:** HTML, CSS, JavaScript
- **Deployment Platform:** Replit Cloud

The use of SQLAlchemy ORM ensures clean and maintainable database interactions, while Flask handles routing, authentication, and business logic.

---

## Tools and Technologies Used

- **Development Tool:** Replit Cloud  
- **Backend Framework:** Flask (Python)  
- **Database:** SQLite (via SQLAlchemy ORM)  
- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** Flask-SQLAlchemy, Werkzeug

## Key Highlights

## Conclusion

SpendSmart demonstrates how Replit Cloud can be effectively utilized for rapid web application development, continuous integration, and real-time deployment. By combining Flask, SQLite, and modern web technologies, the project delivers a practical and scalable solution for managing personal finances in the cloud.
